Midterm Group Project — SweetScoops Ice Cream Shop

This repository contains all the required files for the INFO 102 Midterm Group Project.
The project models a small ice-cream shop using a relational SQL database.

Files Included
	•	midterm_icecream_db.sql — Database schema + inserts + verification queries
	•	midterm_icecream_db.mwb — ERD diagram generated by MySQL Workbench
	•	Final_Midterm_Eda_Reza_All_Objectives.docx — Full documentation including Objectives 1–7, APA references, rubric mapping, and business description

Team
	•	Eda Er (Project Manager) — Objectives 1–4, documentation, formatting, ERD, final review
	•	Reza Safari — Objectives 5–7, SQL design, inserts, verification queries, references

Description

The database includes five connected tables and demonstrates PKs, FKs, UNIQUE, CHECK, and DEFAULT constraints, as well as 1-to-many and many-to-many relationships.
